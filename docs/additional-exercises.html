Dave Armstrong’s {DAMisc} package has a useful function called sumStats that does the same thing using one line of code. As a bonus, it will also give summary statistics by groups, which will come in handy shortly. Unfortunately, it needs an update because one of its dependencies has fallen out of maintenance.

We can also use the base R command t.test().

This option is somewhat limited because it only works when comparing across two categories. However, it does test the significance of the difference, which is useful.

## 
##  Welch Two Sample t-test
